# Diksha Panse - Professional Portfolio

Welcome to the source code for my professional portfolio website! This repository contains a modern, premium, and recruiter-focused web portfolio highlighting my expertise in Backend Engineering, System Design, and AI Automation.

## 🌟 Live Preview
Simply download or clone this repository and open `index.html` in your favorite web browser. No build steps or local servers are required!

## 🚀 Key Features
* **Modern Aesthetic:** A sleek dark theme with glassmorphism effects (frosted glass cards and navigation) inspired by industry leaders like Linear and Vercel.
* **Dynamic Animations:** Features smooth scroll-reveal animations using the AOS library, typing effects, and dynamic number counters.
* **Responsive Design:** Fully responsive layout built with Bootstrap 5 grid systems, ensuring it looks great on mobile, tablet, and desktop screens.
* **Structured Content:** 
  * Comprehensive **Technical Skills** grid.
  * Large, descriptive **Featured Projects** cards (including architecture highlights and tech stacks).
  * Functional mock **Contact Form**.

## 🛠️ Tech Stack
This portfolio is built using lightweight, vanilla web technologies to ensure maximum performance and minimal dependency overhead:
* **HTML5** & **CSS3** (Custom variables, Flexbox, CSS Grid)
* **Vanilla JavaScript** (ES6+)
* **Bootstrap 5** (via CDN for responsive layout)
* **jQuery** (via CDN)
* **AOS (Animate On Scroll)** (via CDN for premium scroll animations)
* **FontAwesome 6** (via CDN for scalable vector icons)
* **Google Fonts** (Inter typeface)

## 📁 Project Structure
* `index.html` - The main structure, semantic layout, and content of the portfolio.
* `index.css` - Custom styling, CSS variables, background gradients, and glassmorphism utilities.
* `index.js` - Client-side logic for animations, scroll-spy navbar, counters, and contact form handling.
* `profile-image.jpg` - Personal profile image used in the hero section.
* `diksha-panse.pdf` - Downloadable resume document.
